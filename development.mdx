<Info>
  **Pré-requisito**: Por favor, instale o Node.js (versão 19 ou superior) antes
  de prosseguir.
</Info>

Passo 1. Instale o Mintlify no seu sistema operacional:

<CodeGroup>

```bash npm
npm i -g mintlify
```

```bash yarn
yarn global add mintlify
```

</CodeGroup>

Passo 2. Vá para onde os documentos estão localizados (onde você pode encontrar `mint.json`) e execute o seguinte comando:

```bash
mintlify dev
```

O site da documentação agora está disponível em `http://localhost:3000`.

### Portas Personalizadas

O Mintlify usa a porta 3000 por padrão. Você pode usar a flag `--port` para personalizar a porta em que o Mintlify é executado. Por exemplo, use este comando para rodar na porta 3333:

```bash
mintlify dev --port 3333
```

Você verá um erro como este se tentar executar o Mintlify em uma porta que já está em uso:

```md
Error: listen EADDRINUSE: address already in use :::3000
```

## Versões do Mintlify

Cada CLI está vinculada a uma versão específica do Mintlify. Por favor, atualize a CLI se o seu site local parecer diferente da produção.

<CodeGroup>

```bash npm
npm i -g mintlify@latest
```

```bash yarn
yarn global upgrade mintlify
```

</CodeGroup>

## Solução de Problemas

Aqui está como resolver alguns problemas comuns ao trabalhar com a CLI.

<AccordionGroup>
  <Accordion title="Mintlify não está carregando">
    Atualize para o Node v18. Execute `mintlify install` e tente novamente.
  </Accordion>
  <Accordion title="Nenhum arquivo ou diretório no Windows">
Vá para o diretório `C:/Users/Username/.mintlify/` e remova a pasta `mint`.
Então, abra o Git Bash neste local e execute `git clone
https://github.com/mintlify/mint.git`.

Repita o passo 3.

  </Accordion>
  <Accordion title="Recebendo um erro desconhecido">
    Tente navegar até a raiz do seu dispositivo e excluir a pasta ~/.mintlify.
    Em seguida, execute `mintlify dev` novamente.
  </Accordion>
</AccordionGroup>
