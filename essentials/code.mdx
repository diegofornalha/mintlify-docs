## Básico

### Código em Linha

Para denotar uma `palavra` ou `frase` como código, coloque-a entre crases (`).

```
Para denotar uma `palavra` ou `frase` como código, coloque-a entre crases (`).
```

### Bloco de Código

Use [blocos de código cercados](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) colocando o código entre três crases e siga as crases iniciais com a linguagem de programação do seu trecho para obter realce de sintaxe. Opcionalmente, você também pode escrever o nome do seu código após a linguagem de programação.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
