O menu de navegação é a lista de links em cada site.

Você provavelmente atualizará o `mint.json` toda vez que adicionar uma nova página. As páginas não aparecem automaticamente.

## Sintaxe de Navegação

Nossa sintaxe de navegação é recursiva, o que significa que você pode criar grupos de navegação aninhados. Você não precisa incluir `.mdx` nos nomes das páginas.

<CodeGroup>

```json Navegação Regular
"navigation": [
    {
        "group": "Introdução",
        "pages": ["quickstart"]
    }
]
```

```json Navegação Aninhada
"navigation": [
    {
        "group": "Introdução",
        "pages": [
            "quickstart",
            {
                "group": "Páginas de Referência Aninhadas",
                "pages": ["pagina-referencia-aninhada"]
            }
        ]
    }
]
```

</CodeGroup>

## Pastas

Basta colocar seus arquivos MDX em pastas e atualizar os caminhos no `mint.json`.

Por exemplo, para ter uma página em `https://seusite.com/sua-pasta/sua-pagina`, você criaria uma pasta chamada `sua-pasta` contendo um arquivo MDX chamado `sua-pagina.mdx`.

<Warning>

Você não pode usar `api` para o nome de uma pasta, a menos que a aninhe dentro de outra pasta. O Mintlify usa o Next.js, que reserva a pasta `api` de nível superior para chamadas internas do servidor. Um nome de pasta como `api-reference` seria aceito.

</Warning>

```json Navegação Com Pasta
"navigation": [
    {
        "group": "Nome do Grupo",
        "pages": ["sua-pasta/sua-pagina"]
    }
]
```

## Páginas Ocultas

Arquivos MDX não incluídos no `mint.json` não aparecerão na barra lateral, mas são acessíveis através da barra de pesquisa e por link direto para eles.
