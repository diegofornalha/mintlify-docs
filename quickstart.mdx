## Configure seu desenvolvimento

Aprenda como atualizar seus documentos localmente e implantá-los para o público.

### Edite e visualize

<AccordionGroup>
  <Accordion icon="github" title="Clone seus documentos localmente">
    Durante o processo de integração, criamos um repositório no seu Github com o
    conteúdo dos seus documentos. Você pode encontrar este repositório no nosso
    [painel de controle](https://dashboard.mintlify.com). Para clonar o
    repositório localmente, siga estas
    [instruções](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository)
    no seu terminal.
  </Accordion>
  <Accordion icon="rectangle-terminal" title="Visualize alterações">
    Visualizar ajuda a garantir que suas alterações estão como pretendido. Nós
    criamos uma interface de linha de comando para renderizar essas alterações
    localmente. 1. Instale o [Mintlify
    CLI](https://www.npmjs.com/package/mintlify) para visualizar as alterações
    da documentação localmente com este comando: ``` npm i -g mintlify ``` 2.
    Execute o seguinte comando na raiz da sua documentação (onde `mint.json`
    está): ``` mintlify dev ```
  </Accordion>
</AccordionGroup>

### Implemente suas alterações

<AccordionGroup>

<Accordion icon="message-bot" title="Instale nosso aplicativo Github">
  Nosso aplicativo Github implanta automaticamente suas alterações no site dos
  seus documentos, então você não precisa gerenciar implantações sozinho. Você
  pode encontrar o link para instalação no seu [painel de
  controle](https://dashboard.mintlify.com). Uma vez que o bot tenha sido
  instalado com sucesso, deve haver uma marca de verificação ao lado do hash do
  commit do repositório.
</Accordion>
<Accordion icon="rocket" title="Envie suas alterações">
  [Faça commit e envie suas alterações para o
  Git](https://docs.github.com/en/get-started/using-git/pushing-commits-to-a-remote-repository#about-git-push)
  para que suas alterações sejam atualizadas no site dos seus documentos. Se
  você enviar e não ver que o aplicativo Github implantou suas alterações com
  sucesso, você também pode atualizar seus documentos manualmente através do
  nosso [painel de controle](https://dashboard.mintlify.com).
</Accordion>

</AccordionGroup>
